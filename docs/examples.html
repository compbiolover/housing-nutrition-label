Case Studies

These examples use the CLI house simulator with real scoring models. Each preset is a different construction profile at the same walkable Cooper-Young location in Midtown Memphis (lat 35.13, lon -89.99). The five construction-driven dimensions are modeled from each build; health, socioeconomic, and walkability are fetched live for the location (so they're identical across presets).

See the interactive nutrition label → — the same construction profiles rendered as a React + D3 label across all nine dimensions.

Score Any US Address

Type any US address and pick a construction profile to generate a live nutrition label for that location — the same nine-dimension scoring the CLI produces. Health & socioeconomic are ranked within the address's own county; energy uses the local climate zone; seismic and flood are pulled for the exact point. Disaster Resilience is the combined expected loss from flood, tornado, earthquake, and fire. Expand Construction details to override the profile with a specific year built, wall type, square footage, and resilience upgrades.

    Location, Location, Location: The Walkability Dimension

    Unlike construction-driven dimensions, walkability is purely a function of where a house sits. The same building — identical resilience, energy, and durability — earns a wildly different walkability grade depending on its neighborhood. These are real Walk Score API results for two actual Shelby County parcels.

    125 Main St, 38103    		 Exurban Eads
    Monterey Woods Cv, 38028

    Downtown: 0.60 × 89 + 0.25 × 54 + 0.15 × 85 = 79.6
    Eads: no transit score → falls back to the raw Walk Score of 0
    Key insight: Shelby County is overwhelmingly car-dependent — 61% of parcels score 0–24 on Walk Score (almost all errands require a car) and not a single parcel reaches "Walker's Paradise" (90+). The walkable pockets that do exist cluster Downtown and in Midtown's older street grid. Because Walk Score is nationally calibrated, the local grade is what surfaces these neighborhoods relative to their car-dependent neighbors: the Downtown parcel earns a national B but a local A — it's good by national standards and the best in the county.
