Interactive Nutrition Label

A React + D3 rendering of the eight scored dimensions as a single at-a-glance label. Pick a construction profile to see how building choices move each dimension. All profiles sit at the same Memphis location (lat 35.15, lon −89.85), so the location-driven dimensions stay fixed while the construction-driven ones change.

Loading visualization…
About this data: Disaster Resilience scores come from the CLI simulator; the remaining dimensions are representative sample values drawn from the published case studies for UI demonstration. Local (percentile) grades require the full county dataset and are omitted here. National grades use the absolute thresholds A ≥ 80, B ≥ 60, C ≥ 40, D ≥ 20, F < 20.
